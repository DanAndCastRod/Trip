# brand_aesthetic

## Rol del documento

Este archivo define el sistema visual de TRIP para web, piezas promocionales, visualizers, portadas y escena.

## Concepto visual

**Mirror Interference**

La estetica de TRIP nace del choque entre reflejo, ruido y desplazamiento. Debe sentirse nocturna, liquida, electrica y ligeramente inestable.

## Principios visuales

- base oscura con acentos frios y neones medidos
- composicion asimetrica y en movimiento
- reflejos, duplicaciones y distorsion como lenguaje, no como filtro gratuito
- tension entre superficie limpia y ruptura visual
- energia de escenario sin caer en barroquismo

## Paleta base

- Void Black: `#09090D`
- Deep Blue: `#1F1659`
- Primary Purple: `#9B44F2`
- Aqua: `#04D99D`
- Cream Sand: `#FFDF92`
- Crimson: `#A82139`

## Reglas de uso del color

- el negro y el azul profundo deben sostener la base
- purpura y aqua funcionan como campo electrico y senal de identidad
- cream sand sirve como respiracion humana dentro del sistema
- crimson se reserva para ruptura, alarma o catarsis
- no usar todos los acentos al mismo tiempo sin una jerarquia clara

## Tipografia

### Logotipo
`California-medium`

### Titulares y golpes visuales
`Revista`

### Cuerpo y UI
`Poppins`

### Detalles, fechas y notas
`rusticprinted`

## UI y composicion

- reticulas fracturadas o desplazadas
- cajas duras combinadas con capas liquidas
- bordes firmes, no blandos
- secciones con sensacion de profundidad y deriva
- evitar layouts excesivamente simetricos cuando el concepto pida quiebre

## Imagen y textura

- siluetas, espejos, cristales, reflejos y doble exposicion
- motion blur, glitch y ruido visual cuando tengan funcion narrativa
- fotografia con alto contraste, luces frias y fondo oscuro
- evitar calidez naturalista o estetica festivalera generica

## Motion

- parallax lento
- desplazamientos flotantes
- pulsos de glitch controlado
- transiciones entre fases del viaje
- evitar rebotes caricaturescos o motion decorativo

## Criterio de validacion

Una pieza visual de TRIP esta bien dirigida si:
- se siente como una travesia nocturna y no como un poster psicodelico generico
- combina introspeccion con energia fisica
- transmite quiebre y movimiento sin perder elegancia
