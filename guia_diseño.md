# Guía de Diseño - Trip

## 1. Paleta de Colores
La identidad visual utiliza paletas contrastantes para reforzar la idea de "viaje" y "dualidad".

### Paleta Principal 1 (Purple/Teal/Cream)
- **Primary Purple:** `#9B44F2`
- **Deep Blue:** `#1F1659`
- **Aqua:** `#04D99D`
- **Forest Green:** `#038C5A`
- **Cream/Sand:** `#FFDF92`

### Paleta Principal 2 (Vibrant Blue/Green)
- **Vibrant Blue:** `#4227F2`
- **Aqua:** `#04D99D`
- **Medium Green:** `#04BF8A`
- **Dark Green:** `#025939`
- **Brown/Earth:** `#AF7245`

### Paleta Complementaria (Warm/Aggressive)
- **Crimson:** `#A82139`
- **Royal:** `#80238C`
- **Violet:** `#732E8F`
- **Electric:** `#883DF2`
- **Bright Red:** `#D92E1E`

## 2. Tipografía
- **Logotipo Principal:** `California-medium` (Estilo pincelada/gestual).
- **Títulos / Impacto:** `Revista` (Estilo Stencil/Industrial).
- **Cuerpo de Texto:** `Poppins` (Bold 700 para jerarquías, Regular para lectura).
- **Detalles / Fechas:** `rusticprinted`.

## 3. Elementos Visuales
- **Conceptos Gráficos:** Siluetas, espejos, reflejos, distorsión líquida.
- **Fondo:** Se sugiere el uso de negro o fondos muy oscuros para resaltar los colores neón/vibrantes.

## 4. Estilo Fotográfico
- **Iluminación:** Contrastes altos, uso de sombras para crear siluetas.
- **Atmósfera:** Introspectiva pero potente.
